# JYp
## JYp简介
1. JYp基于Python所写，有用flask库
2. JYp一个简单的云盘服务端

## 运行命令如下

### Termux运行
```
pkg update
pkg install -y python vim python-pip git
git clone https://github.com/Beiyang5325/JYp
pip install flask
cd JYp
python jyp.py


### 其他linux发行版远行
```
apt update
apt install -y python vim git
git clone https://github.com/Beiyang5325/JYp
pip install flask
cd JYp
python jyp.py
```
